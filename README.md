# iPad Photo Frame

Turn your unused iPad into a digital photo frame with easy wireless photo uploads.

The app exposes a local web server from your iPad, letting you drag and drop photos from any browser on your network.

## Running on Your iPad

1. Open `PhotoFrame.xcodeproj` in Xcode
2. Select your iPad as the run destination
3. You may need to set your Apple ID under **Signing & Capabilities** (a free Apple ID works)
4. Build and run (⌘R)

Once installed, start the web server from Settings to get an upload URL you can access from any device on your WiFi.
